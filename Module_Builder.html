Quick Start: How to Use Module Builder

  1. Basic Info — Create your module, add Learning Outcomes, and set module details.
  2. Covers — Upload a front cover image, fill in the module information table, and optionally add a back cover image.
  3. Introduction Pages — Add your work team members and any introductory text.
  4. Information Sheet — Select a module and learning outcome, then add information sheets with content sections, objectives, and self-check activities.
  5. Activity / Job Sheet — Select a module and learning outcome, then add activity sheets with steps, resources, and criteria.
  6. Assessment Unit — Select a module and add assessment forms with criteria, verification, and signature sections.
  7. References — Add your bibliography entries. These will appear as the last content page before the back cover.
  8. 💾 Save Work — Save your progress to a JSON file at any time. Use Load Work to restore it later.
  9. 📄 Export to Word — When ready, export the complete module as a formatted DOCX file.
